# Twenty Twenty-Four WordPress Theme – Custom Demo

## Description
This is my custom WordPress project using the **Twenty Twenty-Four theme**. I set up and customized the theme using **LocalWP** and hosted it on **InfinityFree**.  

Twenty Twenty-Four is a **flexible, versatile WordPress theme** suitable for any website, including blogs, portfolios, and business sites. It comes with style variations, templates, and full-page designs to speed up site building. Fully compatible with the site editor and new WordPress 6.4+ design tools.

This project demonstrates a working live WordPress website built using the Twenty Twenty-Four theme.

---

## Features
- Flexible and versatile layouts  
- Fully compatible with WordPress 6.4+ site editor  
- Style variations and full-page designs  
- Responsive and mobile-friendly  
- SEO-friendly  
- Compatible with major WordPress plugins and page builders

---

## Tech Stack
- WordPress 6.9+  
- PHP 7.0+  
- MySQL  
- HTML, CSS, JavaScript  

---

## Installation
1. Download or clone this repository.  
2. Copy the `twentytwentyfour` folder to `wp-content/themes/`.  
3. Log in to WordPress Admin → **Appearance → Themes → Activate**.  
4. Optional: Import demo content via **Tools → Import → WordPress**.
